# Bayut Buyer Report

Premium PDF-style buyer report UI built with Next.js, React, and Tailwind CSS.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes

- Each report page is rendered as a centered landscape A4 sheet using a fixed aspect ratio.
- Report content is static mock data and reusable React components.
- Image slots use local placeholder assets under `public/images`.
- The toolbar includes a future-ready Download PDF button placeholder.
